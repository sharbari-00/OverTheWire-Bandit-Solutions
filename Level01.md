# Bandit Level 0 -> 1

### Task
Find the password for the next level, which is stored in a file called readme located in the home directory.

### Commands used:
* **ls (List):** Used to list directory contents to see what files and folders are available in your current location.
* **cat:** Short for "concatenate," used to read and display the text content of a file directly in the terminal.
### Solution:

1. List the files to confirm the readme file exists:
```bash
ls
```
2. Read the file to reveal the password for Level 1:
```bash
cat readme
```


Key concepts Learned
------

* **The Home Directory:** Every user has their own dedicated space to store and manage files.
* **File Inspection:** Using basic commands to navigate and view data inside a Linux environment.
